## 4. 續期通則

```text
1. 哨兵報某把 401/403:先確認服務本身活著(Kuma/portal),再判 token。
2. 續期後:更新 ~/.config/homelab/ 對應 env(600 權限保持)→ 跑
   `hl-token-watch --dry-run` 確認轉綠 → 有下游副本的(MCSM/CF token 等)
   照 V7 對應段同步(portal.env 走 scp+pct push,不走 pct exec echo)。
3. GCal refresh token 重授權:跑 CT260 get_gcal_token.py(OAuth 流程),
   新 refresh_token 寫回 gcal.env。
4. 任何 token 值不進 argv、不進本表、不進 git(gitignore *token*/*secret* 雙保險)。
```
